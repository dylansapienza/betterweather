# Better Weather

A **Modern** **Weather** app for unintrusive and valuable weather insights.

## Features

- Minute Forecasting
- Historical Data
- Moon Phase
- Temperature
- Confidence Rating?
- Weather Email Updates
- Weather Alerts

## Included Packages

**Frontend:**

- React
- Axios

**API:**

- Express
- CORS
- Body-Parser

**Development Tools:**

- Concurrently
- Nodemon
- ESLint
- DotEnv
